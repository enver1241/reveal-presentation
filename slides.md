## Why Reveal.js?  

Reveal.js allows:

- HTML or Markdown slides  
- Live browser-based presentations  
- Themes & transitions  
- Speaker notes  
- Easy GitHub Pages deployment  

---

# 2. What is Reveal.js?

- A modern HTML-based presentation framework  
- Works directly in the browser  
- Supports:  
  - ✔ Markdown  
  - ✔ Images  
  - ✔ Code blocks  
  - ✔ Horizontal & vertical navigation  

--
## Key Features

- Horizontal and vertical slide stacks  
- Theme support (including custom CSS)  
- Plugins for Markdown, highlight, notes, etc.  
- Keyboard and URL-based navigation
