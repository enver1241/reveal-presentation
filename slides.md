## Why Reveal.js? (Vertical Slide)

Reveal.js allows:

- HTML or Markdown slides  
- Live browser-based presentations  
- Themes & transitions  
- Speaker notes  
- Easy GitHub Pages deployment

# 2. What is Reveal.js?

- A modern HTML-based presentation framework  
- Works directly in the browser  
- Supports:  
  - ✔ Markdown  
  - ✔ Images  
  - ✔ Code blocks  
  - ✔ Horizontal & vertical navigation
